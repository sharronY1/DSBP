## 快速开始

### 1. 注册账户

1. 打开DSBP应用
2. 点击"注册"按钮
3. 填写以下信息：
   - **邮箱**: 用于登录和接收邀请
   - **用户名**: 在平台中显示的名称
   - **密码**: 至少8个字符
   - **全名** (可选): 你的真实姓名
4. 点击"注册"
5. 注册成功后自动登录

### 2. 创建第一个项目

1. 登录后，点击"创建项目"按钮
2. 输入项目信息：
   - **项目名称**: 例如 "Web开发项目"
   - **描述** (可选): 项目的简要说明
3. 点击"创建"
4. 项目创建成功，自动进入项目页面

### 3. 创建任务板

1. 在项目页面，点击"创建任务板"
2. 输入任务板名称，例如 "Sprint 1"
3. 点击"创建"
4. **注意**: 免费用户最多创建3个任务板

### 4. 添加第一个任务

1. 在任务板中，点击"添加任务"按钮
2. 填写任务信息：
   - **标题**: 例如 "设计登录页面"
   - **描述** (可选): 详细说明
   - **状态**: 选择 TODO
3. 点击"创建"
4. 任务出现在TODO列中

## 用户管理

### 登录

1. 在登录页面输入邮箱和密码
2. 点击"登录"
3. 登录成功后跳转到项目列表

### 查看个人信息

1. 点击右上角用户头像
2. 选择"个人设置"
3. 查看和编辑个人信息：
   - 用户名
   - 邮箱
   - 全名
   - 许可证类型

### 修改密码

1. 进入个人设置
2. 点击"修改密码"
3. 输入当前密码和新密码
4. 确认修改

## 项目管理

### 创建项目

1. 在项目列表页面，点击"创建项目"
2. 填写项目信息
3. 点击"创建"
4. 项目创建后，你自动成为项目所有者

### 查看项目列表

- 项目列表显示你拥有的项目和参与的项目
- 可以按名称搜索项目
- 可以按创建时间排序

### 编辑项目

1. 进入项目页面
2. 点击项目设置（齿轮图标）
3. 可以修改：
   - 项目名称
   - 项目描述
   - 归档项目

### 删除项目

1. 进入项目设置
2. 点击"删除项目"
3. 确认删除
4. **警告**: 删除项目会删除所有相关的任务板和任务，此操作不可恢复

## 任务板管理

### 创建任务板

1. 在项目页面，点击"创建任务板"
2. 输入任务板名称
3. 点击"创建"

**许可证限制**:
- 免费用户: 最多3个任务板
- 付费用户: 无限制

### 查看任务板

- 任务板显示在项目页面
- 点击任务板进入详情页面
- 可以看到所有任务按状态分组

### 编辑任务板

1. 在任务板页面，点击设置图标
2. 可以修改任务板名称
3. 可以调整任务板顺序

### 删除任务板

1. 进入任务板设置
2. 点击"删除任务板"
3. 确认删除
4. **注意**: 只有项目所有者可以删除任务板

## 任务管理

### 创建任务

1. 在任务板页面，选择要添加任务的状态列（TODO、In Progress或DONE）
2. 点击"添加任务"按钮
3. 填写任务信息：
   - **标题** (必填): 任务名称
   - **描述** (可选): 详细说明
   - **状态**: 自动设置为当前列的状态
   - **分配给** (可选): 选择团队成员
4. 点击"创建"

### 查看任务

- 任务以卡片形式显示在对应的状态列中
- 点击任务卡片查看详情
- 可以看到：
  - 任务标题和描述
  - 创建者和创建时间
  - 分配给谁
  - 最后更新时间

### 编辑任务

1. 点击任务卡片
2. 在任务详情中，点击"编辑"
3. 修改以下信息：
   - 标题
   - 描述
   - 状态（拖拽到不同列或手动选择）
   - 分配给
4. 点击"保存"

### 移动任务状态

有两种方式：

**方式1: 拖拽**
1. 点击并按住任务卡片
2. 拖拽到目标状态列
3. 释放鼠标
4. 任务状态自动更新

**方式2: 编辑**
1. 打开任务详情
2. 在状态下拉菜单中选择新状态
3. 保存

### 删除任务

1. 打开任务详情
2. 点击"删除"按钮
3. 确认删除
4. **注意**: 只有项目成员可以删除任务

### 任务状态说明

- **TODO**: 待办任务，尚未开始
- **In Progress**: 进行中，正在处理
- **DONE**: 已完成

## 团队协作

### 邀请团队成员

1. 进入项目设置
2. 点击"团队成员"标签
3. 点击"邀请成员"
4. 输入要邀请的邮箱地址
5. 点击"发送邀请"
6. 系统会发送邀请邮件

### 接受邀请

1. 收到邀请邮件
2. 点击邮件中的"接受邀请"链接
3. 如果已有账户，直接登录
4. 如果没有账户，先注册
5. 登录后自动加入项目

### 查看团队成员

1. 进入项目设置
2. 查看"团队成员"列表
3. 可以看到：
   - 成员姓名和邮箱
   - 角色（所有者/成员）
   - 加入时间

### 移除成员

1. 进入项目设置
2. 在团队成员列表中找到要移除的成员
3. 点击"移除"
4. 确认操作
5. **注意**: 只有项目所有者可以移除成员

## 许可证管理

### 查看许可证信息

1. 进入个人设置
2. 查看"许可证"部分
3. 可以看到：
   - 当前许可证类型（免费/付费）
   - 已创建的任务板数量
   - 任务板限制

### 免费许可证

**限制**:
- 最多创建3个任务板
- 可以加入无限数量的项目（作为成员）
- 所有基本功能可用

**使用建议**:
- 为每个重要项目创建一个任务板
- 如果项目较多，可以合并相关任务到一个任务板

### 付费许可证

**优势**:
- 无限制创建任务板
- 所有功能可用
- 优先技术支持

**升级方式**:
- 联系管理员
- 或通过支付页面升级

### 任务板计数规则

- 只计算你**拥有**的项目中的任务板
- 作为成员加入的项目中的任务板不计入限制
- 删除任务板后，计数会减少

## 最佳实践

### 项目组织

1. **按项目创建任务板**: 每个项目创建独立的任务板
2. **使用描述**: 为项目和任务板添加清晰的描述
3. **定期归档**: 完成的项目可以归档

### 任务管理

1. **清晰的标题**: 使用简洁明确的任务标题
2. **详细描述**: 在描述中提供足够的信息
3. **及时更新状态**: 任务状态改变时及时更新
4. **合理分配**: 将任务分配给合适的团队成员

### 团队协作

1. **及时邀请**: 项目开始时邀请所有相关成员
2. **明确角色**: 确保团队成员了解自己的角色
3. **定期沟通**: 使用任务描述和评论进行沟通
4. **跟踪进度**: 定期查看任务状态，了解项目进度

### 许可证优化

1. **合并任务板**: 如果接近限制，考虑合并相关任务板
2. **删除无用任务板**: 定期清理不再使用的任务板
3. **升级考虑**: 如果经常需要更多任务板，考虑升级

## 常见问题

### Q: 如何重置密码？

A: 目前需要通过管理员重置，或联系技术支持。

### Q: 可以转移项目所有权吗？

A: 可以，在项目设置中可以将所有权转移给其他成员。

### Q: 删除的任务可以恢复吗？

A: 不可以，删除操作是永久性的，请谨慎操作。

### Q: 如何导出项目数据？

A: 此功能正在开发中，敬请期待。

### Q: 可以自定义任务状态吗？

A: 当前版本固定为三个状态（TODO、In Progress、DONE），未来版本可能支持自定义。
